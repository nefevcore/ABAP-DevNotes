# SAP开发文档

整理下开发经验，主要关注下面几点

- **实施过程常见需求**

    从各类主数据录入，到后勤实施，再到财务结算，沿着数据流转方向，编排知识点。

    > TCODE太多记不住？多利用SAP菜单，主要业务都在其上。

- **示例代码与异常情况分析**

    不同项目，不同需求，少有完全一样的代码。因此该文档只能提供示例代码作为参考，辅以异常情况分析说明，开发人员应当结合实际情况进行调整。

- **技术研究**

    从OP到Cloud，技术更新很多，研究结果也会概况展示。
